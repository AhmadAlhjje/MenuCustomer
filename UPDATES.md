## كيفية الاستخدام

### 1. تشغيل المشروع

```bash
npm run dev
```

### 2. عرض القائمة
- افتح `http://localhost:3000/menu`
- ستظهر جميع المنتجات مع الصور
- يمكنك البحث والتصفية حسب الفئة

### 3. اختيار طبق
1. انقر على زر "اختيار الطبق" في أي منتج
2. ستظهر نافذة منبثقة لتحديد الكمية والملاحظات
3. انقر "إضافة للطلب"

### 4. عرض الطلب
- انقر على زر "طلبك الحالي" في أعلى الصفحة
- ستظهر قائمة جانبية بجميع الأطباق المختارة
- يمكنك تعديل الكمية أو حذف أطباق

### 5. إرسال الطلب
- بعد اختيار جميع الأطباق
- انقر على "إرسال للمطبخ"
- سيتم إرسال الطلب وتفريغ القائمة

## المميزات الفنية

### استخدام التقنيات
- **Next.js 14** - Framework
- **TypeScript** - Type Safety
- **Redux Toolkit** - State Management
- **Tailwind CSS** - Styling
- **React Hooks** - State & Effects

### الصور
- دعم صور متعددة من خلال JSON array
- معالجة روابط الصور بشكل صحيح
- استخدام Next.js Image مع `unoptimized` للصور الخارجية

### التصميم
- Mobile-first responsive design
- Modern UI with gradients and shadows
- Smooth animations and transitions
- RTL support for Arabic

## ملاحظات مهمة

1. **رابط API**: تأكد من أن الـ backend يعمل على `http://localhost:5000`
2. **الصور**: يجب أن تكون الصور متاحة على `/uploads/items/...`
3. **Session**: يجب أن يكون لديك session نشط للطلب
4. **البيانات**: يجب أن يرجع API الصور في شكل JSON array string
